## 디자인 적용 기준

- 한글 음수 자간 금지.
- 긴 문장은 의미 단위로 줄바꿈한다.
- 마지막 줄에 1~3글자만 남는 어색한 줄바꿈을 피한다.
- 기존 냠픽 그린 `#57bf8e` 중심의 톤을 유지한다.
- GIF는 흰 배경이 보이면 사용하지 않고, 누끼 PNG + CSS 모션으로 대체한다.
